# Counts

Counts (also known as *ring down counts*) are the number of positive threshold crossings of a burst signal.

## Parameters

| Name        | Description         | Unit                | Limits        |
|-------------|---------------------|---------------------|---------------|
| `threshold` | Threshold amplitude | Same unit as signal | (0, $\infty$) |

# Variance

The variance is is the squared deviation from the mean.

$$
\begin{aligned}
{Variance} &= \frac{1}{N} \sum_{i=0}^{N-1}{(x[i] - \overline{x})^2} \\
\overline{x} &= \frac{1}{N} \sum_{i=0}^{N-1}{x[i]}
\end{aligned}
$$

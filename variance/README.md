# Variance

The variance is is the squared deviation from the mean.

$$
\begin{aligned}
{Variance} &= \frac{1}{N} \sum_{i = 0}^{N}{(x_i - \overline{x})^2} \\
\overline{x} &= \frac{1}{N} \sum_{i = 0}^{N}{x_i}
\end{aligned}
$$
